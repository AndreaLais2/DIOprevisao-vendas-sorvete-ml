# Previsão de Vendas de Sorvete 🍦

Este projeto foi desenvolvido para prever a quantidade de sorvetes que serão vendidos com base na temperatura do dia.

## Problema
Uma sorveteria deseja prever quantas unidades de sorvete venderá diariamente.
Sabemos que existe uma relação entre temperatura e vendas.
Quanto maior a temperatura, maior tende a ser a venda de sorvetes.

## Solução
Foi treinado um modelo de **Regressão Linear** utilizando Python para prever a quantidade de vendas.

## Tecnologias utilizadas
Python
Pandas
Scikit-learn
Matplotlib
Google Colab

## Resultado
O modelo consegue prever a quantidade de sorvetes vendidos com base na temperatura do ambiente, segundo a meteorologia.

## Exemplo
Temperatura: 35°C
Previsão: 436 sorvetes vendidos
